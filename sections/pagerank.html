Pagerank

"PageRank is[/was] an algorithm used by Google Search to rank websites [...] PageRank works by counting the number and quality of links to a page to determine a rough estimate of how important the website is."

Algorithm (simplified)

  • Iterative algorithm that traverses links
  • Initialize the pagerank for each element with $\frac{1}{\text{#elements}}$
  • If a node $n_1$ links to a node $n_2$, the pagerank contribution of $n_1$ to $n_2$ is computed by $\text{contr}(n_1,n_2) = \frac{\text{pagerank}(n_1)}{\text{#outgoingLinks}(n_1)}$
  • The pagerank of a node $n$ is then $\text{pagerank} (n) = \sum \text{contr}(n_i,n) = \sum \frac{\text{pagerank}(n_i)}{\text{#outgoingLinks}(n_i)}$ where $n_i$ links to $n$
  • Iterative, so we repeat this (until convergence).
